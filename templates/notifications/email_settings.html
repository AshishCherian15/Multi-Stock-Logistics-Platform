Current Status: Email backend is set to console (emails printed to console)
To Enable Real Emails:
  1. Edit greaterwms/settings.py
  2. Uncomment SMTP settings
  3. Add your Gmail credentials
  4. Restart server
# In settings.py, replace:
EMAIL_BACKEND = 'django.core.mail.backends.console.EmailBackend'

# With:
EMAIL_BACKEND = 'django.core.mail.backends.smtp.EmailBackend'
EMAIL_HOST = 'smtp.gmail.com'
EMAIL_PORT = 587
EMAIL_USE_TLS = True
EMAIL_HOST_USER = 'your-email@gmail.com'
EMAIL_HOST_PASSWORD = 'your-16-char-app-password'
Gmail App Password: Use 16-character app password, not regular password. Generate at: Google App Passwords
